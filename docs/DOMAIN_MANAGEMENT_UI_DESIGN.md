# 🎨 域名后台管理系统UI设计文档

## 🎯 设计概述

基于现有的登录页面，设计了完整的域名后台管理系统UI，采用现代化的抽屉布局（Drawer Layout），提供直观、高效的域名管理体验。

## 🏗️ 架构设计

### 布局架构
```
┌─────────────────────────────────────────────────────────┐
│                    Header (固定顶部)                      │
├─────────────┬───────────────────────────────────────────┤
│             │                                           │
│   Sidebar   │            Main Content                   │
│   (抽屉式)   │            (页面内容区)                    │
│             │                                           │
│             │                                           │
└─────────────┴───────────────────────────────────────────┘
```

### 响应式设计
- **桌面端**: 固定侧边栏 (256px宽度)
- **移动端**: 抽屉式侧边栏 (滑动显示/隐藏)
- **平板端**: 自适应布局切换

## 📱 核心组件

### 1. 主布局组件 (`DomainDashboard.tsx`)
- **功能**: 整体布局容器，管理侧边栏状态
- **特性**:
  - 响应式抽屉布局
  - 移动端遮罩层
  - 嵌套路由支持
  - 状态管理

### 2. 侧边栏组件 (`domain-sidebar.tsx`)
- **功能**: 导航菜单和系统功能入口
- **特性**:
  - 分组导航菜单
  - 活跃状态指示
  - 图标 + 文字 + 描述
  - 桌面/移动端适配

#### 导航结构
```
概览
├── 仪表板 - 系统概览和统计信息

域名管理
├── 域名列表 - 查看和管理所有域名
├── DNS记录 - 管理DNS解析记录
└── SSL证书 - 管理SSL证书

统计分析
├── 访问统计 - 域名访问数据分析
└── 性能监控 - 域名性能监控

系统管理
├── 用户管理 - 管理系统用户
├── 操作日志 - 查看系统操作日志
└── 系统设置 - 系统配置和设置
```

### 3. 头部组件 (`domain-header.tsx`)
- **功能**: 顶部导航栏和用户操作
- **特性**:
  - 全局搜索框
  - 通知中心
  - 主题切换
  - 用户菜单
  - 移动端菜单按钮

### 4. 主题切换组件 (`mode-toggle.tsx`)
- **功能**: 亮色/暗色/系统主题切换
- **特性**:
  - 三种主题模式
  - 本地存储持久化
  - 系统主题自动检测

## 📄 页面设计

### 1. 管理员仪表板 (`AdminDashboard.tsx`)
- **布局**: 卡片式网格布局
- **内容**:
  - 统计卡片 (4个核心指标)
  - 最近域名列表
  - 系统状态监控
  - 快速操作面板

#### 统计指标
- 总域名数 (增长趋势)
- 活跃域名 (活跃率)
- 即将过期 (30天内)
- SSL证书 (覆盖率)

#### 功能特性
- 实时数据展示
- 状态徽章指示
- 快速操作入口
- 响应式网格布局

### 2. 域名列表页 (`DomainList.tsx`)
- **布局**: 搜索 + 表格布局
- **功能**:
  - 高级搜索和筛选
  - 分页数据表格
  - 批量操作支持
  - 详细信息展示

#### 表格字段
- 域名名称 (带图标)
- 状态 (徽章显示)
- 注册商
- 过期时间 (倒计时)
- SSL状态
- DNS记录数
- 自动续费状态
- 操作菜单

#### 交互特性
- 实时搜索
- 状态筛选
- 排序功能
- 下拉操作菜单

## 🎨 设计系统

### 色彩方案
- **主色调**: 基于shadcn/ui的现代色彩系统
- **状态色**:
  - 成功: 绿色 (正常状态)
  - 警告: 橙色 (即将过期)
  - 错误: 红色 (异常状态)
  - 信息: 蓝色 (SSL证书等)

### 图标系统
- **图标库**: Lucide React
- **使用规范**:
  - 导航图标: 16px
  - 操作图标: 16px
  - 装饰图标: 20-24px
  - 品牌图标: 32px

### 间距系统
- **基础间距**: 4px的倍数
- **组件间距**: 16px, 24px
- **页面边距**: 24px
- **卡片内边距**: 24px

### 字体层级
```
h1: 30px (页面标题)
h2: 24px (区块标题)  
h3: 20px (卡片标题)
body: 14px (正文)
caption: 12px (辅助文字)
```

## 📱 响应式设计

### 断点设置
```css
sm: 640px   /* 手机横屏 */
md: 768px   /* 平板竖屏 */
lg: 1024px  /* 平板横屏/小笔记本 */
xl: 1280px  /* 桌面显示器 */
2xl: 1536px /* 大屏显示器 */
```

### 适配策略
- **< 1024px**: 抽屉式侧边栏
- **≥ 1024px**: 固定侧边栏
- **网格布局**: 自适应列数
- **表格**: 水平滚动

## 🔧 维护指南

### 添加新页面
1. 在 `src/pages/admin/` 创建页面组件
2. 在 `src/routes/routes.tsx` 添加路由
3. 在 `domain-sidebar.tsx` 添加导航项

### 修改导航菜单
1. 编辑 `domain-sidebar.tsx` 中的 `navigationItems`
2. 添加对应的图标和路由
3. 更新路由配置

### 自定义主题
1. 修改 `tailwind.config.js` 中的色彩变量
2. 更新 CSS 变量定义
3. 调整组件中的样式类名
